# Conversor MD a PDF (UNLaR)
Script para convertir archivos Markdown a PDF con formato institucional de la UNLaR.

## Requisitos
- Python 3.x
- Dependencias: `pip install -r requirements.txt`

## Uso
Ejecutar desde la terminal:
```powershell
cvtmdpdf "archivo.md"
